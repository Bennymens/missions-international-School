# Mission International School Website

This is a static website for Mission International School. To run locally, use a static server like live-server or http-server.

## Structure

- `css/` - All CSS files
- `js/` - All JavaScript files
- `*.html` - Main HTML pages

## Running Locally

1. Install [live-server](https://www.npmjs.com/package/live-server) if not already installed:
   npm install -g live-server

2. Start the server in the project directory:
   live-server .

Then open the provided URL in your browser.
